## Опыт реализации проектов

### Микросервисная платформа для управления умным домом + E-commerce
Спроектировал и разработал комплексное решение, объединяющее управление IoT-устройствами и встроенный маркетплейс.

- Интеграция e-commerce функциональности в существующую инфраструктуру
- Реализация асинхронного взаимодействия микросервисов через Apache Kafka
- Сериализация данных с использованием Apache Avro для обеспечения компактности и совместимости схем

### Рефакторинг сервиса публикации и поиска мероприятий
Выполнил переход от монолитной архитектуры к микросервисной.

- Декомпозиция легаси-кода на независимые доменные сервисы
- Внедрение синхронного взаимодействия через gRPC
- Сокращение времени отклика и повышение отказоустойчивости, что позволило расширить пользовательскую базу

### Backend RESTful приложения для аренды вещей
Разработал полноценный backend для платформы шеринга.

- Реализация бизнес-логики бронирования, валидации запросов и статусов вещей
- Построение гибкой системы фильтрации и пагинации с использованием Spring Data JPA и Hibernate
- Обработка исключений и централизованное логирование

### Система управления банковскими картами
Создал защищённое приложение для работы с финансовыми данными.

- Внедрение JWT-аутентификации и ролевой модели доступа (USER / ADMIN)
- Шифрование чувствительных данных на уровне приложения
- Полностью документированный REST API
